# Retail Data Platform with Agentic AI

This project simulates a real-world retail data platform built using modern data engineering and AI architecture.

## Features

• Retail product data generation using APIs  
• Transaction simulation (POS billing system)  
• AWS S3 data lake (Bronze layer)  
• Data transformation pipeline (Silver layer)  
• Aggregated analytics tables (Gold layer)  
• Agentic AI data agents for querying analytics  
• Interactive business dashboard  

## Architecture

The platform follows a Medallion Architecture:

Bronze → Raw JSON transactions stored in S3  
Silver → Cleaned structured transactional data  
Gold → Business level aggregated analytics  

AI agents query the Gold layer to generate insights.
